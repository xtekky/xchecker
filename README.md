## Requirements
```js
    [WINDOWS]
  * pip install requests
  * pip install pystyle
    [UNIX/LINUX/MACOS]

  * pip3 install pystyle
```

## Installation
```js
    [WINDOWS]
  * pip install -r requirements.txt
  * python xchecker_v2.py
    [UNIX/LINUX/MACOS]
  * pip3 install -r requirements.txt
  * python3 xchecker_v2.py
    
```
